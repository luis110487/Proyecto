# Shop - Sistema de Ventas

Sistema web de ventas desarrollado en Django con funcionalidades de:
- Gestión de productos
- Gestión de clientes  
- Cotizaciones
- Ventas
- Carrito de compras

## Características
- Panel de administración
- Catálogo de productos
- Sistema de cotizaciones
- Gestión de inventario
- Interfaz responsive

## Instalación
1. Clonar el repositorio
2. Crear entorno virtual: `python -m venv venv`
3. Activar entorno: `venv\Scripts\activate` (Windows)
4. Instalar dependencias: `pip install -r requirements.txt`
5. Ejecutar migraciones: `python manage.py migrate`
6. Crear superusuario: `python manage.py createsuperuser`
7. Ejecutar servidor: `python manage.py runserver`
